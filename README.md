# 🏨 Hotel Management Chatbot

A simple and interactive **Hotel Management Chatbot Web Application** built using **Flask, HTML, CSS, and JavaScript**.
This project simulates a basic hotel assistant that can respond to user queries like booking, services, and general information.

---

## 🚀 Features

* 💬 Interactive chatbot interface
* 🏨 Handles hotel-related queries (booking, services, contact, etc.)
* ⚡ Fast and lightweight Flask backend
* 🎨 Clean and responsive UI design
* 🔁 Real-time message exchange between user and bot

## 📁 Project Structure

```
ChatBox/
│── Static/            # CSS, JS, Images
│── HotelM.py          # Flask backend
│── templates/         # HTML files (if added)
│── README.md
```

---

## ▶️ How to Run the Project

1. Clone the repository:
```
git clone https://github.com/Naivete07/Hotel-Management-Chatbot.git
```
2. Navigate to the project folder:
```
cd ChatBox
```
3. Install dependencies:
```
pip install flask
```
4. Run the Flask app:
```
python HotelM.py
```
5. Open browser and go to:
```
http://127.0.0.1:5000/
```
